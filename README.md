# Servo_Motors_Sweep_and_Hold
Arduino project controlling 4 servo motors using the Sweep example for 2 seconds, then holding at 90 degrees
# 4 Servo Motors - Sweep and Hold at 90 Degrees

## Description
This project controls four servo motors using an Arduino Uno in Tinkercad.

The program performs the following actions:
- Runs all four servo motors using the Sweep motion for 2 seconds.
- Stops all servo motors at 90 degrees.

## Components
- Arduino Uno R3
- 4 Servo Motors
- Jumper Wires

## Connections
- Servo 1 → D3
- Servo 2 → D5
- Servo 3 → D6
- Servo 4 → D9

Power:
- Red wire → 5V
- Brown/Black wire → GND
- Orange wire → Signal
